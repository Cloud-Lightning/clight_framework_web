让嵌入式框架
看起来很远，
但用起来很近。

这是一个为 clight framework 设计的概念首页：用冷静的蓝白配色、纯黑文字、 夸张但干净的 3D 卡片结构，把“分层架构、统一 API、平台适配”做成一种一眼就能记住的视觉语言。

查看核心特性 浏览目录关系
API Layer 统一暴露 Gpio / Uart / Timer / Spi / I2c 等公共接口，让业务层优先依赖稳定 API。
Interface Layer 用一致的 C 接口定义底层边界，把平台差异锁进实现层，而不是泄露给业务代码。
Port Layer HPM / STM32 / ESP32 各自对接真实硬件能力，但共享同一套上层调用模型。

核心特性

不是做一个普通的产品落地页，而是把嵌入式框架“分层、映射、平台复用”的逻辑转成有记忆点的视觉首页。

统一 API

清晰调用

把复杂平台差异压缩进统一入口，让业务代码先看到更短、更稳定、更容易记住的调用方式。

3D 视觉

立体层次

通过高对比阴影、厚重投影、悬浮白卡和蓝色景深，形成“远景技术界面”的观感。

蓝白语言

冷静科技

坚持蓝色 + 白色 + 黑字，不用花哨霓虹，靠秩序、光感和卡片厚度做出强识别度。

看起来像未来系统，读起来像技术白皮书。

这一页的设计重点不是装饰，而是让“统一 API、平台接口、底层适配”这些抽象概念，变成有距离感、 但又非常清晰的视觉结构。你一眼就能知道它是做嵌入式框架的，而且是有层次、有边界、有秩序的。
